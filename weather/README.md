# Weather plugin
============

1. Copy files to /etc/cmh-ludl
2. Create Device
..* DeviceName: World Weather
..* D_xxx.xml filename: D_Weather.xml
..* I_xxx.xml filename: I_WUIWeather.xml
3. Reload Vera
4. Edit device
..* ProviderKey
..* Location
..* Metric (1 = Metric, 2 = US Units)
..* EmailDeviceNumber (optional)
5. Reload Veral
6. Update
